Description :

Hi! Vacasky is a creative, elegant, modern, clean and Responsive HTML5/CSS3 Tour & Travel Agency HTML-5 Template based on Bootstrap 5. Compatible with Desktop, Laptop, tablet, mobile or any device. Vacasky is simple and clean design. All files are clearly organized. We believe it will be easy to use and edit them.

Content Mapping :
  • Template root has all .html files and assets.
  • Stylesheets (CSS files) of this template are placed in css folder. It has different css file for different section and page with proper file name.
    All css files imported into style.css file.
  • All Images are in images folder.
  • All Template Javascripts files are in js folder.

HTML Structure

The template is based on Bootstrap 5 the world's most popular front-end component library for developing responsive, mobile-first projects on the web.

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. It includes predefined classes for easy layout options, as well as powerful mixins for generating more semantic layouts.

Create a .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding, then create .row and add columns with appropriate device name (xs, sm, md, lg) and grid number (2, 4, 6, 12).

For full documentation, please visit Bootstrap 5 site: https://getbootstrap.com/docs/5.0/getting-started/download/ 

<section id="" class="">
    <div class="container">
        ...
        ...
    </div>
</section>

Backgrounds

Some of the sections have backgrounds image and to change those backgrounds:

Step 1:

1. Simply open proper css file and find that section class name, change background image by change the link of image directory
OR
2. In that image directory keep your image as same name as like us.
3. How will find proper css file where have to change image?
Ans: Open the file in any browser, then on the section where you have need change, mouse right click and slect inspect.
Then show the css file directory and line number in right side.


For example here showing css for page header background image:

.page_header {
    position: relative;
    background-image: url(../images/background/5.jpg);
    background-position: center;
    background-size: cover;
    background-repeat: no-repeat;
    margin: 0px;
    padding: 150px 0px 150px 0px;
    overflow: hidden;
}
Step 2:

Few section has background image by inline css through html markup. As example:

<section class="feature-one" style="background-image: url(images/background/19.jpg)">
    --------
    --------
</section>

Slider Section

To slide content, we are using popular owlCarousel plugin. And also have swiper plugin, another popular plugin.

Used markup pattern like this:

<section class="banner-one style-two slider-version">
    <div class="single-item-carousel owl-carousel owl-theme">
        <!-- Slide 01 -->
        <div class="slide">
            <!-- Content goes here  -->
        </div>

        <!-- Slide 02 -->
        <div class="slide">
            <!-- Content goes here  -->
        </div>
    </div>
</section>

This markup for OwlCarousel.

<div class="swiper mySwiper2">
	<div class="swiper-wrapper">
		<div class="swiper-slide">
			<!-- Content goes here  -->
		</div>

		<!-- Slide 02 -->
		<div class="swiper-slide">
			<!-- Content goes here  -->
		</div>
	</div>
</div>

This markup for Swiper.

N.B.: Slider control option API in script.js. And you can add/remove/edit api by following Swiper & Owlcarousel slider, mention at bellow.

You can add more slide by following this coding pattern. Then change Content.
